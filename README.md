# C_Hashmap
A *very* simple implementation of a hashmap in the c programming language.

# Goal:
    This project aims to educate about hashmaps, linked lists, and hashing.

# Methods used:
    Hash - DJB2
    Key-Value representation - Separate Chaining
    Buckets - array
